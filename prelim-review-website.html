üêõ Syntax vs Semantic Errors

Key Difference: Syntax errors are caught before running (parsing), semantic errors are caught while running (execution).

üîç Error Detector

JavaScript Syntax Error
// Missing closing brace
function test() {
    console.log("Hello");
// ‚ùå SyntaxError: Unexpected end of input
JavaScript Semantic Error
// Grammar is correct, logic is wrong
let obj = null;
console.log(obj.property);
// ‚ùå TypeError: Cannot read property of null
Python Syntax Error
# Missing colon
if True
    print("Missing colon")
# ‚ùå SyntaxError: invalid syntax
Python Semantic Error
# Grammar is correct, logic is wrong
obj = None
print(obj.property)
# ‚ùå AttributeError: 'NoneType' object has no attribute 'property'

üè∑Ô∏è Typing Systems

JavaScript: Dynamic + Weak (flexible, automatic conversions)
Python: Dynamic + Strong (flexible, explicit conversions)

üß™ Type Coercion Tester

JavaScript (Weak)
"5" + 3        // "53" (string concatenation)
"5" - 3        // 2 (numeric subtraction)
true + 1       // 2 (boolean to number)
[] + []        // "" (empty string!)
Dynamic Typing
let x = 42;        // Number
x = "hello";       // Now string
x = [1, 2, 3];     // Now array
console.log(typeof x); // "object"
Python (Strong)
"5" + 3        # ‚ùå TypeError
int("5") + 3   # ‚úÖ 8 (explicit conversion)
"5" + str(3)   # ‚úÖ "53" (explicit conversion)
Dynamic Typing
x = 42           # Integer
x = "hello"      # Now string
x = [1, 2, 3]    # Now list
print(type(x))   # <class 'list'>

üåê Variable Scope

Scope Chain: Variables are searched from innermost to outermost scope - Block ‚Üí Function ‚Üí Global

üîç Scope Visualizer

JavaScript Scope Levels
let global = "Everyone sees me";     // üåç Global

function myFunction() {
    let funcScope = "Function only";   // üè† Function
    
    if (true) {
        let blockScope = "Block only"; // üì¶ Block
        console.log(global);          // ‚úÖ Can access
        console.log(funcScope);       // ‚úÖ Can access  
        console.log(blockScope);      // ‚úÖ Can access
    }
    
    console.log(blockScope);          // ‚ùå Error!
}
Variable Hoisting
console.log(hoistedVar);  // undefined (not error!)
var hoistedVar = "I'm hoisted";

console.log(hoistedLet);  // ‚ùå ReferenceError
let hoistedLet = "Temporal dead zone";
Python Scope (LEGB Rule)
global_var = "Everyone sees me"     # üåç Global

def my_function():
    func_var = "Function only"       # üè† Local
    
    def nested_function():
        nested_var = "Nested only"   # üì¶ Local
        print(global_var)           # ‚úÖ Can access
        print(func_var)             # ‚úÖ Enclosing scope
        print(nested_var)           # ‚úÖ Can access
    
    print(nested_var)               # ‚ùå NameError!
Global & Nonlocal Keywords
counter = 0

def modify_global():
    global counter
    counter += 1  # Modifies global

def outer():
    x = 10
    def inner():
        nonlocal x
        x += 1    # Modifies enclosing scope
    return inner
